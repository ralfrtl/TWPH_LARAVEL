# Git
```
git clone https://github.com/ralfrtl/TWPH_LARAVEL.git
```
```
git checkout <branch>
```
# Laravel
```
cp .env.example .env
```
```
composer install
```
or
```
composer update
```

# Node
```
npm install
```

# Artisan
```
php artisan migrate
```
```
php artisan db:seed --class=<Seeder>
```
```
php artisan key:generate
```
```
php artisan jwt:secret
```

# Start Project
```
php artisan serve
```
```
npm run dev
```
